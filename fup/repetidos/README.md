# Contar repetidos

![_](../../.tko/cache/e2bc1de5ac270eaa50476053dc6bb1f9ff79f7ba/base/repetidos/assets/cover.jpg)

### Entrada

- 1a linha: O número P(Procurado) e N(Tamanho do vetor).
- Nas próximas N linhas, cada linha contém um número inteiro representando um elemento do vetor.

### Saída

- A quantidade de vezes que o número P aparece no vetor.

## Exemplos

<!-- load tests.toml --tests 2 -->
```py
>>>>>>>> INSERT
5 3
2
3
6
======== EXPECT
0
<<<<<<<< FINISH
```

```py
>>>>>>>> INSERT
0 4
0
0
1
0
======== EXPECT
3
<<<<<<<< FINISH
```
